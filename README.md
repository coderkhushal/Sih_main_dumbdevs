### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/coderkhushal/sih_internal_backend
   ```
2. Navigate to the project directory: 
   ```bash
   cd sih-internal-frontend
   ```
   
3. Install the dependencies:
   ```bash
   npm install
   ```
   
4. Run the project:
   ```bash
   npm run dev
   ```
