### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/coderkhushal/Sih_main_dumbdevs/frontend
   ```
2. Navigate to the project directory: 
   ```bash
   cd Sih_main_dumbdevs
   ```
   
3. Install the dependencies:
   ```bash
   npm install
   ```
   
4. Run the project:
   ```bash
   npm run dev
   ```
